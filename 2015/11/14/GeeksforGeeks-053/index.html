GeeksforGeeks-053-纯虚函数和抽象类

By Xue Wentao

发表于 2015-11-14

有的时候我们无法实现所有基类提供的函数，这样一个类叫做抽象类。
例如，令Shape成为一个抽象类，我们不能够提供draw()函数的实现，但是我们知道每一个派生类都必须实现draw()函数。
C++中的纯虚函数是一个不需要实现的虚函数。只需要声明它。
纯虚函数在声明时设为0。

C++ VS Java
在Java中，一个类可以通过使用abstract关键字来定义为抽象类。同样，一个函数也可以通过abstract关键字来定义为纯虚或抽象的

接口 VS 抽象类
接口不会实现任何函数，它被认为是方法声明的集合。
在C++中，接口可以通过让一个类的所有函数都是纯虚函数来模拟
