虚函数的用途
虚函数允许我们创建多个基类指针，并且调用任何派生类的方法，而不需要知道派生类的种类。

编译器是如何实现的？
如何实现：虚表和虚指针

1. vtable-一张包含了所有函数指针的表，每个类都有
2. vptr-指向vtable的指针，每个对象有一个

编译器会在两个地方增加代码以便维护和使用vptr：

1. 在每个构造函数里面。vptr被创建，并指向vtable
2. 多态函数调用时。当一个多态调用被创建时，编译器插入代码去查找vptr，当获取了vptr时，派生类的vtable会被获取，查找vtable，获取对应派生类函数的地址。

GeeksforGeeks-037-C++中子类可以从父类中继承什么

By Xue Wentao

发表于 2015-10-18

可以被继承的：

1. 每一个在父类中定义的数据成员，即使不能被派生类访问
2. 每一个在父类中定义的普通函数，即使不能被派生类访问
3. The same initial data layout as the base class（继承父类的初始化列表）

不会被继承的：

1. 基类的构造和析构函数
2. 基类的友元

static关键字可以被用于局部变量，函数，类的数据成员和对象。static局部变量只会被初始化一次，并且在函数调用之间保存其值。static函数可以通过范围解析操作符直接被调用。
